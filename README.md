## ⚡ Quick Local Setup

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Setup Environment**: Create a `.env` file with your `MONGO_URI` and `SECRET_KEY`.
3. **Run the app**:
   ```bash
   python app.py
   ```
4. **Access UI**: Go to [http://127.0.0.1:5000](http://127.0.0.1:5000)
